# Design System

This document outlines the design principles, color palette, and typography for the site. Whether you are a human contributor or an AI agent, please adhere to these guidelines to maintain a consistent industrialist aesthetic.

## Philosophy
The site follows an **industrialist, minimalist aesthetic**. Design is intended to be functional, structured, and deliberate. Color is a tool for emphasis—not decoration—and should be used sparingly.

## Color Palette

The site is built on a neutral base of pure black and white. Flare colors are assigned per-page to guide user attention and provide high-impact emphasis.

### Base Colors
| Role | Hex |
| :--- | :--- |
| Background | `#FFFFFF` |
| Text/Elements | `#000000` |

### Flare Colors
Flare colors are to be used only for interactive elements, key highlights, or subtle structural accents.

| Page | Flare Color | Hex |
| :--- | :--- | :--- |
| **Landing** | Pumpkin Spice | `#FF6700` |
| **Resume** | Medium Jungle | `#2BA84A` |
| **Gallery** | Cobalt Blue | `#004BA8` |

## Typography

We utilize a mono-dominant hierarchy that balances structural weight with technical precision.

### Font Stack
* **Title / Display:** `JetBrains Mono` (Extra Bold, 800)
    * *Usage:* Primary headers, H1–H3, hero title, stat values.
* **Subtitle / Interface:** `Inter` (Regular, 400)
    * *Usage:* Navigation labels, section tags, project metadata, tags, marquee, UI labels.
* **Body:** `JetBrains Mono` (Light, 200)
    * *Usage:* Long-form copy, project descriptions, about paragraphs.

*Note: This design system is currently under active development. If you are modifying CSS or UI components, ensure your changes respect the hierarchy and constraints outlined above.*
