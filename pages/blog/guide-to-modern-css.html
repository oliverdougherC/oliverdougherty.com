A Guide to Modern CSS

July 22, 2024

CSS has evolved far beyond simple styling. Modern CSS provides powerful layout, animation, and theming capabilities without JavaScript.

CSS Custom Properties

Custom properties (CSS variables) enable dynamic theming and cascading values that can be updated at runtime through the cascade, media queries, or JavaScript.

"CSS is the only programming language I've enjoyed writing. It's declarative, visual, and the feedback loop is instant."

Modern Layout

Grid and Flexbox have replaced float-based layouts entirely. Grid excels at two-dimensional layouts while Flexbox is ideal for one-dimensional distributions.

Theming Example

A simple theme system using custom properties:

:root {
    --color-primary: #fed0bb;
    --color-bg: #0a0a0a;
    --font-body: "Inter", sans-serif;
}

[data-theme="light"] {
    --color-primary: #2a2a2a;
    --color-bg: #ffffff;
}

Combined with @media (prefers-color-scheme), dark mode becomes a single attribute toggle.
