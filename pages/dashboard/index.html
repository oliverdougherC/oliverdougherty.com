    Retro VM

    Idle

    Launch a fresh local session when you are ready. Nothing persists after the tab closes.

    V1 brief

    Desktop-first, fully client-side, and intentionally isolated. The guest boots on demand and starts fresh every time.

    Guest
    Tiny Core Linux 11 · 20 MB remastered retro ISO
    Session
    Ephemeral per tab · clean boot every launch
    Bridge
    Clipboard paste only · offline-first rollback
    Actions

    Desktop browser recommended. Click into the VM screen to capture the mouse. Press Escape to release it, or to exit fullscreen. This Tiny Core rollback is offline-first.

    Scope guardrails
    • No server-side compute.
    • No saved writable state after reset or tab close.
    • Guest internet is intentionally deferred in this rollback.
    • Designed for keyboard + mouse desktops first.
